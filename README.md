# Hotel Management Project

## Overview
A Java-based console application for managing hotel operations including room booking, customer management, and billing. This project demonstrates object-oriented programming concepts, file handling, and a real-world application structure.

--- 

## Features
- Room Management: Add, update, and view rooms.
- Customer Management: Register, view, and manage customer information.
- Booking System: Check room availability and handle bookings.
- Billing System: Generate bills for customers.
- File Handling: Save and retrieve data from files for persistence.
- Console-Based UI: Simple and intuitive command-line interface.
